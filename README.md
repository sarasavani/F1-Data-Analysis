# Formula 1 Race Performance Analysis

## Overview

This project is an interactive Python-based tool for analyzing and comparing Formula 1 race performance using historical lap-by-lap data. It focuses on transforming raw race data into meaningful performance metrics and visualizations to support exploratory and comparative analysis of drivers and races.

## Features

- Selection of drivers and races for analysis  
- Lap-by-lap performance comparison  
- Race pace and consistency analysis  
- Position tracking throughout a Grand Prix  
- Visualization of lap times and race dynamics  
- Statistical comparison between drivers  

## Methodology

The project implements a full data analysis pipeline:

- Data loading from CSV files  
- Data cleaning and preprocessing  
- Feature extraction and transformation  
- Statistical analysis of lap times and positions  
- Visualization of performance trends over time  

## Tools & Technologies

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- CSV data processing

## Purpose

The goal of this project is to explore Formula 1 race dynamics through data analysis, focusing on lap time behavior, consistency, and race progression patterns.
